Instructions to Run (Google Colab)

Open Google Colab
Create a new notebook
Copy and paste the code from AI_Dashboard.ipynb
Run the cell (Shift + Enter)
Use the interactive filters to see live updates
Take screenshots of the output
Screenshots All screenshots are available in the screenshots/ folder.
